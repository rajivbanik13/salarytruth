# SalaryTruth — Free Salary Tools for Indian Professionals

Live site: https://salarytruth.in

## Tools
- **Payslip Decoder** — Verify PF, HRA, TDS instantly
- **CTC to In-Hand Calculator** — Exact take-home after all deductions
- **Tax Regime Comparison** — Old vs new regime for your salary
- **Offer Letter Analyzer** — Red flag detection
- **Notice Period Calculator** — Buyout cost calculator
- **Salary Negotiation Script** — Data-backed negotiation
- **20+ Salary Breakdowns** — ₹3 LPA to ₹1 Crore

## Tech Stack
- Pure HTML, CSS, Vanilla JS
- No framework, no build process
- 100% browser-based — no data sent to server
- Hosted on Hostinger

## Privacy
All calculations run in your browser.
Zero tracking. Zero login. Zero data collection.
